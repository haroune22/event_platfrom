# Event Platform

A fullstack community platform where users can:

- create communities
- share posts
- organize events
- upload learning resources
- comment and interact with content

## Tech Stack

### Backend

- Node.js
- Express.js
- MySQL
- JWT Authentication
- Raw SQL Queries

### Frontend

- React
- React Query
- Context API / Redux
- Tailwind CSS

## Features

- Authentication with JWT & Cookies
- Community system
- Posts & comments
- Events & event attendees
- Learning resources
- Saved posts
- User interests system
- Feed filtering by category
- Pagination & infinite scroll (planned)
- Rate limiting (planned)

## Database Design

The project uses a relational MySQL database.

ERD available in `/docs`.

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
